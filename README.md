# Word Bank Game 

A Python word guessing game where you guess words from a hidden word bank.

##  How to Play

- The program randomly selects a word from a list
- You guess letters one by one
- For each wrong guess, you lose a try
- Guess all letters correctly before running out of tries to win!

##  How to Run

1. Make sure Python is installed on your computer

2. Download the file or clone the repository

3. Open terminal/command prompt in the folder

4. Run the game:
   ```bash
   python word_bank_game.py
